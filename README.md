# Ravity

Ravity

## Installation

Use the package manager [yarn](https://classic.yarnpkg.com/lang/en/docs/install/#windows-stable) to install Ravity.

```bash
yarn
```

## Usage

# important

make sure to copy **.env.example** and paste it into **.env** file (**you should create one**)
**_REACT_APP_SERVICE_URL_** : we have here BASE URL variable for service (**you should add the url service here**)
PS : all the api calls begin with '/' (' https://xxx.xxxxxx.xx/xxx') !!!!

# run Project

yarn start

# build Project

yarn build
